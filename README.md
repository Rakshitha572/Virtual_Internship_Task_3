# 📊 Adidas Sales Performance & Business Intelligence Dashboard

## 📌 Project Overview
This project focuses on performing **Deep-Dive Analysis and Interactive Dashboarding** on the Adidas Sales Dataset.  
The objective is to analyze sales performance, identify business trends, and create an interactive dashboard for better decision-making.

The project was completed as part of a **Virtual Internship Task**.

---

# 🎯 Objectives
- Perform Exploratory Data Analysis (EDA)
- Analyze sales trends and profitability
- Identify top-performing products and regions
- Create KPI-based business insights
- Build an interactive Power BI dashboard

# 📊 Dashboard Features
## ✅ KPI Cards
Displays overall business performance metrics.

## ✅ Regional Sales Analysis
Analyzes sales contribution across different regions.

## ✅ Product Performance Analysis
Identifies top-performing products based on revenue.

## ✅ Monthly Sales Trend
Shows sales growth and seasonal patterns over time.

## ✅ Interactive Filters / Slicers
Allows dynamic filtering by:
- Region
- Product
- Retailer

# 🔍 Analysis Performed

## 📌 Exploratory Data Analysis (EDA)
- Data cleaning
- Missing value analysis
- Descriptive statistics
- Outlier detection

## 📌 Business Analysis
- Sales trend analysis
- Profitability analysis
- Product performance evaluation
- Regional contribution analysis

## 📌 Customer Segmentation
K-Means clustering was used to identify customer purchasing behavior patterns.

---

# 📈 Key Insights
- Footwear products generated the highest sales revenue.
- West region contributed the maximum sales.
- Sales showed strong seasonal trends.
- Higher sales directly impacted profitability.
- Major retailers significantly influenced total revenue.
